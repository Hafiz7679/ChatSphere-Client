# рҹҡҖ Overview

ChatSphere Client is the frontend of the ChatSphere real-time messaging platform.

It provides a modern, responsive user interface with secure authentication, instant messaging, online user tracking, typing indicators, profile management, and a premium user experience.

The application communicates with the Express backend through REST APIs and Socket.IO for real-time communication.

# вҡҷпёҸ Installation

## Clone Repository

```bash
git clone https://github.com/Hafiz7679/ChatSphere-Client.git

cd ChatSphere-Client
```

---

## Install Dependencies

```bash
npm install
```

---

## Create Environment File

Create

```
.env
```

Add

```env
REACT_APP_API_URL=http://localhost:5000/api
REACT_APP_SOCKET_URL=http://localhost:5000
```

---

## Run Development Server

```bash
npm start
```

Runs the app on

```
http://localhost:3000
```
